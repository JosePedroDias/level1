# Command Line Interface


Use the Command Line Interface to experiment with the API and work fast.



Fire the CLI:

    node run_level1_cli.js mydb



Use it...

    » clear

    » add {"a":"b"}

    » put a {"c":"d", "e":[2,4]}

    » list

    » keys

    » values

    » get a

    » del a

    » search "a" in v
